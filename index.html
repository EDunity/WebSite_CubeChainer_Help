[ 前書き ]
このゲームは、ゲーム開始時に個人のアカウントを作成するので、いつでもインターネット環境さえあればゲームモード別の世界ランキングで自分の順位を確認することが出来ます。


[ 作者について ]
Unityインターハイ2019でゴールドアワードを受賞した高校3年生です。 毎日Unityを触っているので「毎日Unity」という名前でやっています。 モデリングとシナリオを考えるのが少し苦手です。

・ゲームの始め方・



[ アカウントを持っている場合 ]
まずSignInを押してSignIn画面に移動してください。 次に以前作成したPlayerNameとPassWordを入力し、SignInを押してSignInを完了してください。 入力したPlayerNameまたはPassWordが間違っているとSignInを完了することが出来ません。 SignInが完了するとタイトル画面に移動します。 タイトル画面に移動したらGameStartを押してゲームを開始してください。 インターネット環境がないとSignInは完できないので注意してください。


[ アカウントを持っていない場合 ]
まずSignUpを押してSignUp画面に移動してください。 次にPlayerNameとPassWordを決め、入力しSignUpを押してSignUpを完了してください。 入力したPlayerNameが既に他のプレイヤーに使用されていた場合、SignUpを完了することは出来ません。 SignUpが完了するとSignIn画面に移動します。 SignIn画面では、先ほどSignUp画面で入力したPlayerNameとPassWordを入力し、SignInを押してSignInを完了してください。 SignInが完了するタイトル画面に移動します。 タイトル画面に移動したらGameStartを押してゲームを開始してください。 インターネット環境がないとSignUp/SignInは了完了できないので注意してください。

・ゲームの遊び方・



ゲームを開始すると、141種類あるブロックの内のいずれかが落下してきます。 これは決してランダムではなく、全プレイヤーを公平にするための法則を考え適応させています。 落下してくる全てのブロックは、落下し終わると形が崩れます。 プレイヤーは、落下してくるブロックを操作し、同色のブロック同士をつなげて左右の壁を横断させることでブロックが消去され、スコアを稼ぐことができます。 ブロックを消した量によって得られるスコアは変動します。 さらにこのゲームには様々な難易度のスキルが存在し、ゲーム中にこれらのスキルを繰り出すことによってボーナススコアを稼ぐことができます。 難しいスキルになればなるほど得られるスコアは高くなります。 ブロックを赤いライン以上積み上げてしまうとゲームオーバーです。

・GameModeについて・



[ Normal ]
ブロックの種類は141種類で落下速度は普通です。10000スコア毎にブロックの落下速度が速くなります。このゲームモードが基本です。




[ HighSpeed ]
Normalのブロックの落下速度が速くなったバージョン。とても難易度が高く、初心者にはお勧めできません。




[ Retro ]
Normalのブロックの種類が21種類になったバージョン。スキルも組みやすいので練習するならこのGameModeがおすすめ！

・Skillについて・



[ AllClear ]
フィールド上のキューブを全て消すたびに500のボーナススコアが得られます。一番簡単なSkillです。




[ Combo ]
1つのブロックを設置することでキューブを連鎖的に消去するたびに1000のボーナススコア得られます。
ボーナススコアが得られるのは、2Comboからなので注意してください。




[ Ren ]
ブロックを設置するたびにキューブを消去すると、その都度750のボーナススコア得られます。
ボーナススコアが得られるのは、2Renからなので注意してください。




[ Double ]
2色のキューブを同時に消去するたびに2000のボーナススコア得られます。
かなり難しいSkillです。




[ Triple ]
3色のキューブを同時に消去するたびに3000のボーナススコア得られます。
難易度が高い分、得られるボーナススコアも多いです。

・Rankingについて・



4つの世界ランキングを用意しました。 ゲームモード別のランキングが3つとプレイヤーのレベルのランキングが1つです。 ゲームモード別のランキングでは、全プレイヤーの各ゲームモードでの最高スコアをスコアの高い順に表示します。 プレイヤーのレベルのランキングでは、全プレイヤーのプレイヤーレベルの順位をレベルの高い順に表示します。 インターネット環境がないとランキングは見れないので注意してください。

・Customizeについて・



3つのカスタマイズを用意しました。 GameFieldとCubeSkinとCubeColorの3つです。 GameFieldではフレームの有無やゴーストブロックなどのゲーム中での細かい設定を変更できます。 CubeSkinではゲーム中のキューブの模様を変更出来ます。 キューブの模様は7種類の中から1つ洗濯してください。 CubeColorではゲーム中のキューブの色を変更できる出来ます。 キューブの色は８種類の中から3つ選択してください。 カスタマイズには、プレイヤーレベルの制限があるので注意してください。
