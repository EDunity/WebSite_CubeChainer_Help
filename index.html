・ゲームの始め方・



[ 初めてプレイする場合 ]
ゲームを始める前にSignUpとSignInをする必要があります。 手順は次の通りです。 まず、SignUpを押してSignUp画面に移動します。 次に、PlayerNameとPassWordを決め、入力しSignUpを押してSignUpを完了します。 入力したPlayerNameが既に他の人に使用されていた場合、SignUpを完了できません。 SignUpが完了するとSignIn画面に移動します。 次に、SignIn画面では、先ほどSignUp画面で入力したPlayerNameとPassWordが入力されているので、SignInを押してSignInを完了します。 SignInが完了するタイトル画面に移動します。 最後に、タイトル画面に移動したらGameStartを押してゲームを開始します。 インターネット環境がないとSignUp/SignInは完了できません。


[ 初めてではない場合 ]
ゲームを始める前にSignInする必要があります。 手順は次の通りです。 まず、SignInを押してSignIn画面に移動します。 次に、以前作成したPlayerNameとPassWordが入力されているので、SignInを押してSignInを完了します。 入力したPlayerNameまたはPassWordが間違っているとSignInを完了できません。 SignInが完了するとタイトル画面に移動します。 最後に、タイトル画面に移動したらGameStartを押してゲームを開始します。 インターネット環境がないとSignInは完了できません。

・ゲームの遊び方・



ゲームを開始すると、数百または数十種類あるブロックの内のいずれかが落下してきます。 これは決してランダムではなく、全プレイヤーを公平にするための法則を考え適応させています。 落下してくる全てのブロックは、落下し終わると形が崩れます。 プレイヤーは、落下してくるブロックを操作し、同色のブロック同士をつなげて左右の壁を横断させることでブロックが消去され、スコアを稼ぐことができます。 ブロックを消した量によって得られるスコアは変動します。 さらに、このゲームには様々な難易度のスキルが存在し、ゲーム中にこれらのスキルを繰り出すことによってボーナススコアを稼ぐことができます。 難しいスキルになればなるほど得られるスコアは高くなります。 ブロックを赤いライン以上積み上げてしまうとゲームオーバーです。

・GameModeについて・



[ Normal ]
「Normal」は、ブロックの種類は141種類で、10000スコアを稼ぐ度にブロックの落下速度が速くなるゲームモードです。このゲームモードが一般的です。




[ HighSpeed ]
「HighSpeed」は、ブロックの種類は141種類で、ブロックの落下速度が極端に速いゲームモードです。非常に難易度が高いので、上級者におすすめです。




[ Retro ]
「Retro」は、ブロックの種類が21種類だけで、10000スコアを稼ぐ度にブロックの落下速度が速くなるゲームモードです。ブロックの種類が少なくてスキルが組みやすいので、初心者にはこのゲームモードがおすすめ！

・Skillについて・



[ AllClear ]
フィールド上のキューブを全て消すと500のボーナススコアが得られるスキルです。
最も簡単なスキルです。




[ Combo ]
1つのブロックを設置することでキューブを連鎖的に消去する度に1000のボーナススコアが得られるスキルです。
ボーナススコアが得られるのは、2回目のComboからなので注意です。




[ Ren ]
ブロックを設置する度にキューブを消去すると、その都度750のボーナススコアが得られるスキルです。
ボーナススコアが得られるのは、2回目のRenからなので注意です。




[ Double ]
2色のキューブを同時に消去すると2000のボーナススコアが得られるスキルです。
かなり難しいスキルです。




[ Triple ]
3色のキューブを同時に消去すると3000のボーナススコアを得られるスキルです。
難易度が高い分、得られるボーナススコアも多いです。

・Rankingについて・



4つの世界ランキングを用意しました。 ゲームモード別のランキングが3つとプレイヤーのレベルのランキングが1つです。 ゲームモード別のランキングでは、全プレイヤーの各ゲームモードでの最高スコアをスコアの高い順に表示します。 プレイヤーのレベルのランキングでは、全プレイヤーのプレイヤーレベルの順位をレベルの高い順に表示します。 世界ランキング上位を目指してください！ インターネット環境がないとランキングは見れないので注意です。

・Customizeについて・



3つのカスタマイズ要素を用意しました。 「GameField」と「CubeSkin」と「CubeColor」の3つです。 「GameField」では、フレームの有無やゴーストブロックなどのゲーム中での細かい設定を変更できます。 「CubeSkin」では、ゲーム中のキューブの模様を変更できます。 キューブの模様は7種類の中から1つを選択してください。 「CubeColor」では、ゲーム中のキューブの色を変更できます。 キューブの色は８種類の中から3つを選択してください。 カスタマイズには、プレイヤーレベルの制限があるので注意です。
